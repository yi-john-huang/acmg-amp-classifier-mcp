## Demonstration Components

### Standard Structure

Each demonstration includes:

1. **Clinical Case Presentation** (2-3 minutes)
   - Patient background
   - Clinical presentation
   - Testing indication
   - Family history

2. **AI Agent Interaction** (8-15 minutes)
   - Natural language query
   - Real-time MCP tool calls
   - Evidence gathering display
   - Classification process

3. **Results Review** (3-5 minutes)
   - Final classification
   - Clinical implications
   - Treatment recommendations
   - Family counseling points

4. **Q&A Session** (5-10 minutes)
   - Audience questions
   - Technical discussion
   - Clinical pearls
   - Alternative scenarios
